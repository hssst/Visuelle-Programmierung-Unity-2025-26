# Visuelle-Programmierung-Unity-2025-26
Projekt immer mit aktuellem Datum hochladen.

Ablauf für jede Person:

Einmalig klonen
GitHub Desktop öffnen → „Clone a repository“
Euer Repo auswählen → lokalen Ordner wählen
Unity-Projekt liegt dann dort.

Arbeiten
In Unity Scripts/Szenen ändern.

Änderungen hochladen
GitHub Desktop zeigt geänderte Dateien automatisch.
Häkchen setzen → Commit Message schreiben → Commit → Push origin klicken.

Änderungen der anderen holen
GitHub Desktop → Fetch origin / Pull.
